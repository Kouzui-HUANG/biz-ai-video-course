# 輸出協議：人物設定聖經

### 階段二：《人物設定聖經》正式文稿

#### A. 人物總覽 (Character Executive Summary)
*(150-200字，精準總括角色的「靈魂輪廓」、初見印象與核心內在張力。)*

#### B. 細節資料 (High-Resolution Character Data)

**1. 實體建構 (Physical Entity)**
* **姓名**：[全名/暱稱/稱號]
* **基本參數**：[年齡] / [性別] / [身高] / [體重]
* **面部與微表情**：[眼神特質、五官特徵及特定情緒下的微表情]
* **外顯風格**：[髮型/日常穿衣風格與標誌性配件，需反映其心理狀態]

**2. 心理與性格矩陣 (Psychological Matrix)**
* **性格素描**：[3-5 個核心形容詞，並附帶具體情境下的行為體現]
* **人格標籤**：[MBTI及專屬子類型解釋] / [星座] / [九型人格或血型]
* **價值觀與底線**：[絕對捍衛的事物 / 觸發暴怒或崩潰的雷區]

**3. 社會與文化根基 (Socio-Cultural Roots)**
* **身分背景**：[國籍/族裔/信仰與價值觀影響]
* **原生家庭**：[家庭結構及對其性格/創傷的具體影響]
* **語言能力**：[母語/流利度/特定口音或階級用語特徵]

**4. 語言與行為特徵 (Linguistic & Behavioral Fingerprint)**
* **對話口吻**：[如：玩世不恭帶點尖酸、溫和但充滿距離感等]
* **慣用語癖**：[3-5 個常掛在嘴邊的詞彙、語氣詞或特定句型]
* **潛意識動作**：[焦慮、說謊、或高度專注時的無意識肢體動作]

**5. 專業軌跡 (Professional Trajectory)**
* **職涯背景**：[現任職務與塑造能力的過往核心經歷]
* **能力邊界**：[專業強項與致命盲區]
* **溝通受眾 (TA)**：[最容易共鳴的群體，以及最常起衝突的群體]

**6. 隱藏維度 (Hidden Dimensions)**
* **不為人知的秘密**：[僅有自己或極少數人知道的習慣或往事]
* **發展潛力與毀滅隱患**：[未來故事中可能使其昇華或墮落的關鍵節點]
