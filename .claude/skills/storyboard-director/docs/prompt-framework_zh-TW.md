# 提示詞工程框架

## 結構公式

在認知循環的路徑 B 或 C 中生成最終英文提示詞時，嚴格遵循以下結構公式以確保最佳 AI 圖像生成品質：

```
[主體細節] + [場景/環境] + [攝影技術] + [光影與氛圍] + [藝術風格]
```

### 組件分解

| 組件 | 說明 | 範例 |
|---|---|---|
| **主體細節** | 外貌特徵、服裝、姿勢、表情、道具 | "銀白髮高馬尾的年輕女性，穿深藍軍裝大衣，握劍柄，銳利目光" |
| **場景/環境** | 地點、建築、年代、天氣、時間 | "站在中世紀歐洲小鎮的雨後鋪石巷道，傍晚時分" |
| **攝影技術** | 景別、攝影角度、鏡頭類型、景深 | "中近景，仰角，35mm 鏡頭，淺景深" |
| **光影與氛圍** | 光源、品質、情緒、體積光效果 | "背後的戲劇性輪廓光，體積霧，暖琥珀色街燈光暈，電影氛圍" |
| **藝術風格** | 渲染風格、藝術參考（不使用品牌名稱） | "2D 動畫風格，高度精細賽璐璐上色，飽和度高" |

## 景別參考

| 景別 | 中文 | 說明 |
|---|---|---|
| Extreme Wide Shot (EWS) | 大遠景 | 建立場景，主體在畫面中非常小 |
| Wide Shot (WS) | 遠景 | 完整環境，主體可見 |
| Full Shot (FS) | 全景 | 主體從頭到腳，包含部分環境 |
| Medium Wide Shot (MWS) | 中遠景 | 主體從膝蓋以上 |
| Medium Shot (MS) | 中景 | 主體從腰部以上 |
| Medium Close-Up (MCU) | 中近景 | 主體從胸部以上 |
| Close-Up (CU) | 近景 | 臉部與肩膀 |
| Extreme Close-Up (ECU) | 大特寫 | 單一特徵（眼睛、手、物品） |
| Over-the-Shoulder (OTS) | 過肩鏡頭 | 越過一個角色看向另一個角色 |

## 攝影角度參考

| 角度 | 中文 | 效果 |
|---|---|---|
| Eye Level | 平視 | 中性、具親和力 |
| Low Angle | 仰角 | 權力、支配感、宏偉 |
| High Angle | 俯角 | 脆弱、渺小 |
| Bird's Eye | 鳥瞰 | 全知俯視 |
| Dutch / Tilted | 荷蘭角 / 傾斜 | 緊張、不安、失衡 |
| Worm's Eye | 蟲視 | 極度權力感、高聳壓迫 |

## 鏡頭類型參考

| 鏡頭 | 效果 |
|---|---|
| 廣角 (16-35mm) | 開闊感、輕微變形、戲劇性透視 |
| 標準 (35-50mm) | 自然、最接近人眼 |
| 望遠 (85-200mm) | 壓縮景深、隔離主體、散景 |
| 微距 | 對小物件的極致細節 |
| 變形寬銀幕 | 電影感光暈、更寬畫面比例感 |

## 光影詞彙

* **主光 (Key Light)**：主要光源，定義主要陰影方向。
* **輪廓光 / 背光 (Rim / Back Light)**：輪廓邊緣光暈，將主體與背景分離。
* **補光 (Fill Light)**：軟化主光產生的陰影。
* **體積光 (Volumetric Light)**：透過霧/塵/靄可見的光線。
* **環境光 / 漫射光 (Ambient / Diffused Light)**：柔和、無方向性光（陰天、室內反射光）。
* **場景光源 (Practical Light)**：場景中的光源（燈具、霓虹燈、蠟燭）。
* **明暗對比法 (Chiaroscuro)**：高對比戲劇性光影。
* **黃金時刻 (Golden Hour)**：日出/日落時暖色調、低角度陽光。
* **藍色時刻 (Blue Hour)**：冷色調、柔和的黎明/黃昏光線。

## 構圖指南

* **三分法 (Rule of Thirds)**：將關鍵元素置於三分線交叉點。
* **引導線 (Leading Lines)**：利用環境線條引導觀者視線。
* **框中框 (Frame within Frame)**：利用門、窗、拱門框住主體。
* **留白 (Negative Space)**：刻意留出空白區域以營造情感重量。
* **前景層次 (Foreground Interest)**：以前景元素層疊創造景深。

## 對白整合規則

如使用者提供對白、旁白或字幕：
* 在提示詞描述中保留其 **原始語言**。
* 自然地融入對白上下文：例如 "the character shouts「別過來！」with a raised hand."
* 不要將對白翻譯為英文置入提示詞中。
