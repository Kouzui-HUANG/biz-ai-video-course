# Prompt Engineering Framework

## Structural Formula

When generating the final English prompts (in Path B or C of the Cognitive Loop), strictly adhere to the following structural formula to ensure optimal AI image generation quality:

```
[Subject Details] + [Setting/Environment] + [Cinematography] + [Lighting & Atmosphere] + [Art Style]
```

### Component Breakdown

| Component | Description | Examples |
|---|---|---|
| **Subject Details** | Features, clothing, pose, expression, props | "A young woman with silver-white hair in a high ponytail, wearing a dark blue military coat, gripping a sword hilt, intense gaze" |
| **Setting/Environment** | Location, architecture, era, weather, time of day | "Standing on a rain-soaked cobblestone alley in a medieval European town at dusk" |
| **Cinematography** | Shot size, camera angle, lens type, depth of field | "Medium close-up, low-angle shot, 35mm lens, shallow depth of field" |
| **Lighting & Atmosphere** | Light source, quality, mood, volumetric effects | "Dramatic rim lighting from behind, volumetric fog, warm amber streetlamp glow, cinematic mood" |
| **Art Style** | Rendering style, artistic reference (no brand names) | "2D animation style, highly detailed cel-shading, rich color saturation" |

## Shot Size Reference

| Shot Type | Chinese | Description |
|---|---|---|
| Extreme Wide Shot (EWS) | 大遠景 | Establishes location, subject is tiny in frame |
| Wide Shot (WS) | 遠景 | Full environment with subject visible |
| Full Shot (FS) | 全景 | Subject head-to-toe, some environment |
| Medium Wide Shot (MWS) | 中遠景 | Subject from knees up |
| Medium Shot (MS) | 中景 | Subject from waist up |
| Medium Close-Up (MCU) | 中近景 | Subject from chest up |
| Close-Up (CU) | 近景 | Face and shoulders |
| Extreme Close-Up (ECU) | 大特寫 | Single feature (eye, hand, object) |
| Over-the-Shoulder (OTS) | 過肩鏡頭 | Looking past one character to another |

## Camera Angle Reference

| Angle | Chinese | Effect |
|---|---|---|
| Eye Level | 平視 | Neutral, relatable |
| Low Angle | 仰角 | Power, dominance, grandeur |
| High Angle | 俯角 | Vulnerability, smallness |
| Bird's Eye | 鳥瞰 | Omniscient overview |
| Dutch Angle / Tilted | 荷蘭角 / 傾斜 | Tension, unease, disorientation |
| Worm's Eye | 蟲視 | Extreme power, towering presence |

## Lens Type Reference

| Lens | Effect |
|---|---|
| Wide-angle (16-35mm) | Expansive, slight distortion, dramatic perspective |
| Standard (35-50mm) | Natural, closest to human eye |
| Telephoto (85-200mm) | Compressed depth, isolates subject, bokeh |
| Macro | Extreme detail on small objects |
| Anamorphic | Cinematic lens flares, wider aspect ratio feel |

## Lighting Vocabulary

* **Key Light**: Primary light source, defines dominant shadow direction.
* **Rim / Back Light**: Silhouette edge glow, separates subject from background.
* **Fill Light**: Softens shadows from key light.
* **Volumetric Light**: Visible light rays through fog/dust/haze.
* **Ambient / Diffused Light**: Soft, non-directional (overcast sky, interior bounce light).
* **Practical Light**: In-scene light sources (lamps, neon signs, candles).
* **Chiaroscuro**: High-contrast dramatic lighting (light vs. dark).
* **Golden Hour**: Warm, low-angle sunlight during sunrise/sunset.
* **Blue Hour**: Cool, diffused twilight light.

## Composition Guidelines

* **Rule of Thirds**: Place key elements along third-line intersections.
* **Leading Lines**: Use environmental lines to guide viewer's eye.
* **Frame within Frame**: Use doors, windows, arches to frame subject.
* **Negative Space**: Deliberate empty areas for emotional weight.
* **Foreground Interest**: Layer depth with foreground elements.

## Dialogue Integration Rule

If the user provides dialogue, voiceover, or subtitles:
* Retain them in their **original language** within the prompt description.
* Place dialogue contextually: e.g., "the character shouts「別過來！」with a raised hand."
* Do NOT translate dialogue into English within the prompt.
