## 1. 色彩與材質底層原理

**60-30-10 層級**——色彩不是裝飾，是空間情緒的結構。提示詞中的顏色必須按層級陳述：

| 層級 | 承載者 | 占比 |
|---|---|---|
| 主色 | 牆、地、天花、大片櫃體 | 約 60% |
| 輔色 | 沙發、窗簾、地毯、木皮、石材 | 約 30% |
| 點綴色 | 抱枕、畫作、花器、單椅、燈具 | 約 10% |

提示詞句型：`dominant warm off-white walls and pale oak flooring, a secondary layer of sage-green sofa and oatmeal linen curtains, accented only by terracotta cushions and one brass floor lamp`（主色暖米白牆與淺橡木地板，輔色鼠尾草綠沙發與燕麥色亞麻簾，僅以陶土色抱枕與一盞黃銅立燈點綴）。

**材質＝空間的體感**——木頭給溫度、石材給重量、金屬給銳度、玻璃給穿透、布料給柔軟、皮革給成熟感、水泥給原始感。材質的色彩、紋理與表面處理（霧面／磨面／拋光）和油漆同等地決定氛圍。

**風格＝一致性的控制，不是貼標籤**——成熟空間在以下面向維持單一邏輯：線條語言、光線色溫、金屬色、木色、家具比例。三段提示詞中重複這些錨點。

## 3. 各風格執行要訣

讓每種風格在生成時不翻車的一行修正：

- **現代風**：不能只做白灰黑（會像空屋）——加木色、布料、局部石材給溫度。
- **極簡風**：極簡＝視覺噪音少，不是東西少——低飽和色、精準收邊、隱藏收納；必須指定光線品質與比例，否則變冷清。
- **北歐風**：不是「白牆＋木地板」——地毯、植栽、桌燈、掛畫才是完成度關鍵；要有生活感。
- **日式風**：節制、水平線、真實材質；最怕假符號堆疊——與其堆「和風裝飾」，不如控制留白、光線與材料真實感。
- **侘寂**：是經過整理的樸素，不是破舊——每處粗糙材質旁都要有安靜的留白。
- **Japandi**：木色統一成單一色族；黑色只能少量點綴，否則柔和變沉重。
- **工業風**：住宅工業風必須軟化——皮革、地毯、暖光、木頭去中和水泥與鋼鐵的冰冷。
- **美式風**：高級感來自比例與層次，絕不是買大件家具就完成。
- **法式風**：避免過甜——霧面材質、一件現代家具或深色木頭讓空間更成熟。
- **新古典**：線板比例要根據牆面尺度設計；金色過量顯俗。
- **Art Deco**：最適合做局部主題（玄關、餐廳、酒櫃牆、主臥），不必全屋鋪滿。
- **Mid-century**：靈魂在家具輪廓——沒有經典型體，光靠顏色不成立。
- **Boho**：要有收斂底色——牆地天花保持米白或土色，花紋才顯豐富而非混亂。
- **地中海**：抽取灰泥質感、拱形、陶土色即可——避免全套藍白變主題餐廳。
- **飯店風**：關鍵是燈光分層——環境光、間接光、閱讀燈、情境燈要分開指名。

## 4. 材質語彙

| 材質 | 氣質 | 適合風格 | 注意事項 |
|---|---|---|---|
| 淺橡木 | 清新、自然 | 北歐、Japandi、日式 | 避免搭過黃的燈光 |
| 胡桃木 | 成熟、穩重 | 美式、Mid-century、飯店風 | 小空間用太多會重 |
| 大理石 | 奢華、冷冽 | 新古典、Luxury、飯店風 | 紋路要控制，不可太亂 |
| 洞石 | 柔和的自然奢華 | 侘寂、現代、地中海 | 多孔材質——寫磨面，不寫拋光 |
| 水泥／微水泥 | 原始、冷靜 | 極簡、工業、侘寂 | 必須用溫暖材質平衡 |
| 黃銅 | 復古、精緻 | Art Deco、法式、新古典 | 少量才高級，大量顯俗 |
| 黑鐵 | 硬朗、俐落 | 工業、現代、現代東方 | 避免全室過暗 |
| 藤編 | 輕盈、度假感 | 北歐、Boho、日式 | 點綴用，不宜過量 |
| 絲絨 | 華麗、柔軟 | 法式、Art Deco、Luxury | 深色才有質感，淺色絲絨顯廉價 |
| 亞麻 | 樸素、自然 | 侘寂、日式、北歐 | 易皺——把皺感寫成特色 |

## 5. 風格混搭公式

| 混搭方向 | 成功公式 |
|---|---|
| 現代＋日式 | 白牆、木格柵、低家具、細黑色細節 |
| 北歐＋日式 | 淺木、米白、圓角家具、紙燈 |
| 工業＋現代 | 微水泥、黑鐵、皮革、簡潔俐落線條 |
| 美式＋現代 | 線板、深座舒適沙發、乾淨平面櫃體 |
| 法式＋現代 | 線板背景、現代家具、節制的黃銅 |
| 侘寂＋飯店風 | 粗質灰泥牆、溫暖分層燈光、精緻寢具 |
| Art Deco＋現代 | 幾何線條、黃銅點綴、一張深色單椅、安靜背景 |

規則：**一個主風格、至多兩個副風格**——主風格擁有主色層級與家具輪廓的決定權。

## 6. 紀律規則

把經典室內設計錯誤轉成提示詞硬規則：

1. **設計整體框架，不是單品**——一張漂亮椅子不等於漂亮空間；每段提示詞都陳述完整 60-30-10 層級。
2. **主材最多 3–5 種**——指名之後就停手。
3. **一室一個木色家族**——淺木、黃木、紅木、深木隨意混用顯廉價。
4. **一個主導金屬色**——第二種金屬只能是刻意的少量點綴。
5. **光線色溫必須配風格**——暖風格（侘寂、鄉村、美式、Boho、飯店）用 2700–3000K 暖光；冷風格（極簡、工業、現代）可用中性白。暖風格配冷白光直接毀掉空間——永遠指名色溫。
6. **禁止符號堆疊**——日式≠到處掛和風簾；法式≠貼線板了事；工業≠只有裸管。風格由色盤＋材質＋輪廓的一致性建立。

## 7. 風格驅動設計流程

需求是風格驅動（室內空間，非戲劇場景）時，依專業判斷順序建立設計聖經：

1. **生活型態**——屋主怎麼吃飯、休息、收納、工作？（化為生活痕跡）
2. **情緒需求**——安靜、華麗、溫暖、俐落，還是有故事感？
3. **主風格**——一主、至多兩副（見 §5 混搭公式）
4. **色彩板**——主色／輔色／點綴色，按 60-30-10（§1）
5. **材質板**——從 §4 選 3–5 種，一個木色、一個金屬色
6. **家具輪廓**——§2 該風格的線條語言
7. **家飾最後**——畫、燈、地毯、抱枕、花器，取自該風格家飾語彙

**迷你風格聖經範例（Japandi 客廳）**：主風格 Japandi · 主色暖米白牆＋淺橡木地板 · 輔色灰米亞麻沙發＋藤編休閒椅 · 點綴少量霧黑金屬＋一只深色陶花器 · 材質：淺橡木、棉麻、紙、藤、霧面陶 · 光：2700K 紙立燈＋薄紗簾透入的柔日光 · 輪廓：低矮、圓潤、懸浮感。
