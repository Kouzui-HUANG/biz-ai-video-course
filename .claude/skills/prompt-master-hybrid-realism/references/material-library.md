# Material Library — Texture Vocabulary

| Material | High-Fidelity Tags |
|----------|-------------------|
| Wool / Knit | detailed knitted wool texture, visible yarn fibers, cable-knit pattern, soft fuzzy surface |
| Silk / Satin | glossy silk sheen, liquid-smooth satin drape, subtle light ripples, specular highlights on folds |
| Denim | indigo denim weave, visible weft threads, faded whiskers, raw selvedge edge |
| Cotton | soft matte cotton, fine woven texture, natural fabric creases |
| Leather | rich leather grain, embossed texture, warm patina, stitched seams |
| Lace | intricate lace filigree, delicate mesh pattern, scalloped edges |
| Velvet | plush velvet nap, crushed velvet light play, color-shift pile direction |
