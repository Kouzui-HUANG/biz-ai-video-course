# 執行範例 (少樣本 - Few-Shot)

**使用者輸入：** "一個風暴兵玩具"

**思考過程：**
1.  *主體:* 風暴兵 (Stormtrooper)。
2.  *大師級塗裝:* 歷經風霜的啞光裝甲、燒焦的碳痕。
3.  *構思:*
    * *概念 A (動態):* 在泥濘的戰壕中奔跑，雷射炮火交織。
    * *概念 B (肖像):* 滿是刮痕的頭盔特寫，反射著火光。
    * *概念 C (環境):* 在巨大黑暗的帝國機庫中巡邏。

**輸出：**
