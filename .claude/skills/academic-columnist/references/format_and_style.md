# Format and Style Guidelines

## 1. Jargon Reduction Protocol
- Identify highly specialized terms (e.g., "Blasiertheit", "Teleological Paradox").
- Immediately substitute them with intuitive concepts (e.g., "冷靜的自我保護", "手段與目的倒置").
- If a technical term must be kept for impact, pair it instantly with a vivid, everyday metaphor.

## 2. Structural Requirements
The final article must be approximately 1200 words, strictly divided into 5 paragraphs:
- **Paragraph 1 (The Hook):** Describe a modern pain point. Seamlessly introduce the 3 core viewpoints.
- **Paragraphs 2, 3, 4 (The Body):** Dedicate one paragraph to each core viewpoint. Develop the argument logically.
- **Paragraph 5 (The Synthesis):** A strong, resonant final thought leaving the reader with an actionable paradigm shift.

## 3. Formatting
- **Main Title:** Catchy and shareable, wrapped in `【】` (e.g., `【當世界淪為賭場：現代人的三層心智防線】`).
- **Subtitles:** Each of the 5 paragraphs must start with a sharp, one-sentence subtitle prefixed with `▋ `.
- **Paragraphs:** Add line breaks within each of the 5 main sections to improve readability. Do not output massive walls of text.
- **Clean Output**: Output ONLY the article text. No conversational filler.

## 4. Tone
Professional, empathetic, and profound. Emulate a top-tier magazine columnist. Use Traditional Chinese (zh-TW).
