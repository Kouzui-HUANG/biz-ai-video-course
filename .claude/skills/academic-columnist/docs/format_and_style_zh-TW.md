# 格式與風格指南

## 1. 術語削減協議 (Jargon Reduction Protocol)
- 辨識高度專業的術語（例如：「冷淡態度 (Blasiertheit)」、「目的論悖論」）。
- 立即使用直觀的概念進行替換（例如：「冷靜的自我保護」、「手段與目的倒置」）。
- 若必須保留技術術語以維持張力，請務必緊跟著一個生動、日常的譬喻來解釋它。

## 2. 結構要求 (Structural Requirements)
最終文章必須約為 1200 字，並嚴格分為五個段落：
- **第一段（破題）**：描述一個現代社會的痛點，並無縫引入即將探討的三個核心觀點。
- **第二、三、四段（主體）**：將每一個核心觀點分配至獨立段落。邏輯清晰地展開論述。
- **第五段（總結）**：強而有力的最終迴響，留給讀者一個具備行動力的思想典範轉移。

## 3. 排版格式 (Formatting)
- **主標題**：吸睛且具備分享價值，並使用 `【】` 包裹（例如：`【當世界淪為賭場：現代人的三層心智防線】`）。
- **段落標題**：五個段落的開頭都必須有一句銳利、一針見血的段落標題，並加上 `▋ ` 符號。
- **段落留白**：在這五個主要區塊內，針對不同的語意進行適當的換行（分段），避免出現難以閱讀的文字牆。
- **乾淨輸出**：最終的輸出必須只包含專欄文章本身，請勿加上任何前言或結語。

## 4. 語氣 (Tone)
專業、具備同理心且思想深刻。模仿頂級雜誌專欄作家的文風。全程使用繁體中文 (zh-TW) 撰寫。
