## 1. 具體—普世階梯

### 核心原則

**越具體，反而越普世。**

抽象詞讓所有人都能說，但沒有人會記住；具體畫面只屬於敘述者，但讀者反而能對號入座。

### 階梯範例（從差到好）

| 階梯 | 範例 | 評價 |
|---|---|---|
| L1：抽象情緒詞 | 「我很傷心」 | ❌ 空話，人人能說 |
| L2：類別化情境 | 「失戀很痛」 | ⚠️ 仍抽象，無記憶點 |
| L3：通用畫面 | 「眼淚一直流」 | ⚠️ 太常見，無辨識度 |
| L4：具體物件 | 「妳留下的牙刷我留了三個月」 | ✅ 只屬於我，但人人懂 |
| L5：物件 + 動作 + 時間 | 「妳的牙刷在杯子裡 我每天看它三秒才出門」 | ✅✅ 頂級具體 |

### 三條鐵則

1. **能寫動作就不寫形容詞**
   - ❌「我很憤怒」 → ✅「我把門摔到牆都裂了」
   - ❌「她很冷漠」 → ✅「她吃飯時眼睛只看碗」

2. **能寫物件就不寫概念**
   - ❌「我們的回憶」 → ✅「妳忘在副駕的口紅」
   - ❌「青春」 → ✅「制服第二顆鈕扣」

3. **能寫畫面就不寫感受**
   - ❌「孤獨」 → ✅「外送來了 我跟外送員多聊了三句」
   - ❌「想念」 → ✅「我把你的名字打進搜尋欄 又一個字一個字刪掉」

### 訓練法：物件寫作（Object Writing）

> Pat Pattison 在 Berklee 音樂學院教了 30 年的核心訓練法

- **計時 10 分鐘**
- **隨機選一個具體名詞**（雨傘、門把、咖啡漬、舊鑰匙、便利商店發票⋯）
- 用「七感」全方位描寫（詳見[§6 七感感官調用](#6-七感感官調用)）
- 不求精煉、不求文采，只求**具體**
- **每天 1 次，連做 90 天**，建立你的「感官記憶資料庫」

**輸出時直接調用資料庫**：寫詞時不必憑空編，從累積的感官筆記裡挑現成材料拼接即可。

## 2. 鉤子優先（Hook First）

### 核心原則

專業作詞 80% 從這裡開始：**先確立副歌核心金句（Hook），所有主歌都為了「把聽眾帶到這一句」而寫。**

### Hook 的判定標準

一句話如果同時滿足以下 4 點，才算合格的 Hook：

1. **可複述性**：陌生人聽一次就記得
2. **情緒指向性**：一聽就知道這首歌在講什麼情緒
3. **可演唱性**：旋律高點落在最重要的字上
4. **延展性**：能放在歌名、副歌反覆、結尾收束三個位置都不違和

### Hook 的四種類型（對應 SKILL.md 的 Q2 ABCD）

#### A. 直白告白型
- 口語化、情緒赤裸
- 例：「越過山丘 才發現無人等候」（李宗盛〈山丘〉）
- 例：「我不要你離開 距離隔不開 思念變成海」（梁靜茹〈勇氣〉的反向版本）
- **語法特徵**：主詞 + 動詞 + 受詞，句構簡單

#### B. 詩意隱喻型
- 意象豐富、文學感重、留白多
- 例：「有時候 有時候 我會相信一切有盡頭」（王菲〈紅豆〉/ 林夕詞）
- 例：「時間是讓人猝不及防的東西」（楊宗緯〈洋蔥〉/ 阿信詞）
- **語法特徵**：常用「⋯是⋯」「⋯像⋯」「⋯讓⋯」的判斷句式

#### C. 敘事畫面型
- 強畫面感、像在說故事的單一鏡頭
- 例：「天青色等煙雨 而我在等你」（周杰倫〈青花瓷〉/ 方文山詞）
- 例：「車站人擁擠 我不能說話」（陶喆〈月亮代表誰的心〉）
- **語法特徵**：時間 + 地點 + 主角動作，類似電影單鏡頭

#### D. 哲思反思型
- 抽離視角、帶哲學重量、常用反問
- 例：「為何舊知己 在最後變不到老友」（陳奕迅〈最佳損友〉/ 黃偉文詞）
- 例：「最熟悉的陌生人 還能做什麼」（蕭亞軒〈最熟悉的陌生人〉）
- **語法特徵**：常用「為何」「怎麼」「還能」等疑問或反問詞

### Hook 製作工序

```
步驟 1：寫 Premise（核心情境，1-2 句白話）
步驟 2：產出 5-10 句候選 Hook
步驟 3：用「四項判定」篩出 1-2 句
步驟 4：唸出來確認可演唱性
步驟 5：把這句放副歌第一行
步驟 6：主歌反向設計，鋪陳這句的「之前」與「之後」
```

## 3. 韻腳張力管理

### 核心原則（Pat Pattison 韻腳系統）

**完美韻 = 穩定 = 結束感**；**不完美韻 = 不穩定 = 想繼續聽**。

**鋪陳用不完美韻吊著、收尾用完美韻落地。**

### 韻腳類型分級

| 類型 | 中文範例 | 英文範例 | 張力 | 用途 |
|---|---|---|---|---|
| **完美韻（Perfect Rhyme）** | 光/亡、家/花、心/林 | light/night | 穩定、封閉 | 副歌收尾、最終句 |
| **家族韻（Family Rhyme）** | 光/狂、心/沉 | light/time | 中等、半穩 | Pre-chorus、段落內 |
| **加綴韻（Additive Rhyme）** | 心/心理、家/家鄉 | rain/range | 微不穩 | 製造變化 |
| **減綴韻（Subtractive Rhyme）** | 心理/心、家鄉/家 | range/rain | 微不穩 | 製造變化 |
| **母音韻（Assonance）** | 光/方、心/今 | light/find | 不穩、開放 | 主歌、鋪陳 |
| **子音韻（Consonance）** | 心/新、光/剛 | light/late | 不穩 | 內韻、句中 |

### 中文作詞的韻腳實務

#### A. 韻部使用建議（普通話發音）

- **開口韻**（a、ang、an）：情緒外放、適合副歌爆發
  - 例：光、方、揚、長、亡、忘
- **閉口韻**（i、in、ing）：情緒內斂、適合主歌內省
  - 例：心、深、沉、林、銀、行
- **撮口韻**（u、ü）：圓潤、適合溫柔慢板
  - 例：路、霧、土、訴、樹
- **複合韻**（ai、ei、ao、ou）：中性、可前可後
  - 例：海、雨、後、走

#### B. 主韻 + 變韻原則

- 一首歌**不可整首使用同一個韻母**
- 主歌、副歌、橋段**至少使用 2-3 個不同韻部**
- 例：主歌用閉口韻（in/ing），副歌轉開口韻（ang）製造情緒對比

#### C. 五大禁忌

1. **撞韻**：同韻字重複出現太密（每句末字都是「光、亡、忘」）
2. **擠韻**：同字或近音字短距離重複（「天涯」與「無涯」並列）
3. **倒字**：旋律高點落在輕聲或不重要的字（會破壞演唱性）
4. **硬湊韻**：為了押韻使用不通順的詞序或冷僻詞
5. **死韻**：副歌反覆使用完美韻，聽起來像兒歌或廣告

### 張力布局範本

```
[verse 1]    家族韻（半穩定，吊著）
[pre-chorus] 不完美韻（製造懸念）
[chorus]     完美韻（落地、記憶點）
[verse 2]    家族韻或加綴韻（變化但連貫）
[bridge]     換韻部！（製造對比）
[chorus]     回到原韻部完美韻（回家感）
[outro]      母音韻（開放、留白）
```

## 4. 詞曲咬合（Prosody）

### 核心原則

**重音必須落在強拍上，最重要的字必須在旋律高點或長音上。**

**唸出來不順的句子，唱出來絕對更糟。**

### 中文 Prosody 的三層檢查

#### 第一層：節拍對齊

- 自然語言重音 = 旋律重拍
- 例：「我**愛**你」中的「愛」必須落在強拍
- 反例：旋律強拍若落在「我」上，會聽成「**我**愛你」（強調對比），扭曲原意

#### 第二層：聲調對齊（中文特有）

- 中文有四聲，旋律走向應與聲調趨勢一致或互補
- **上揚旋律** 配 二聲（陽平，如「來」）或 三聲後段（如「好」）
- **下降旋律** 配 四聲（去聲，如「愛」）
- **長音 / 高音** 避免配 輕聲字（會破壞語意）

**經典案例**：方文山〈青花瓷〉「素**胚**勾勒出青花」 —— 「胚」字（一聲）落在旋律高音長音，發音飽滿。若改為四聲字「**碎**胚」會極不協調。

#### 第三層：呼吸點對齊

- 樂句斷句必須匹配語意斷句
- 不要把「我們的故事」斷成「我們/的故事」
- 不要在虛詞（的、了、嗎、呢）後面斷氣

### 驗證法：三遍唸讀法

1. **第一遍**：用平常說話的節奏唸（不要假裝唱）
   - 卡頓的地方 → Prosody 有問題
2. **第二遍**：刻意把每句重音強調出來
   - 重音落在虛詞 → 改寫
3. **第三遍**：閉眼唸，只憑聽覺判斷
   - 拗口、繞口、撞音 → 修

### Prosody 與 Suno 的關係

Suno 會根據歌詞自動生成旋律。歌詞 Prosody 寫得好，Suno 生成的旋律才會自然。**Prosody 就是「寫給 Suno 看的隱形樂譜」**。

- 重要字寫成 2-3 音節短語（讓 Suno 知道這裡要拉長）
- 情緒高點放在句末（讓 Suno 在這裡放高音）
- 避免長句一氣呵成（Suno 會卡氣口）

## 5. 視角切換推進

### 核心原則

**視角的移動本身就是情緒推進的引擎。**

一首歌全程第一人稱會悶、全程第二人稱會變說教、全程第三人稱會冷漠 —— 視角切換才能做出層次。

### 標準視角結構（4 種主流模式）

#### 模式 A：經典三段推進（最常用）

```
主歌 1：第一人稱（我的處境、我看到什麼）
主歌 2：第二人稱（對你說、你怎麼了）
副歌：抽離視角（我們、這個世界、第三人觀察）
橋段：時空切換（過去 / 未來 / 夢境）
```

範例邏輯：
- V1：「我坐在那家咖啡店 點了你愛的拿鐵」（第一人稱現在式）
- V2：「妳是不是也會 偶爾想起這條街」（第二人稱詰問）
- 副歌：「有些人 就是註定 要在記憶裡老去」（第三人稱普世）
- 橋段：「如果時間倒回 那年夏天 我們還在傘下」（時空閃回）

#### 模式 B：書信體（單向視角，信件感）

- 全程第二人稱「你」
- 適合：道別、告白、回憶單一對象
- 範例：陶喆〈Susan 說〉

#### 模式 C：雙視角對話（高難度）

- 男女視角輪流出現
- 適合對唱、戲劇感強的敘事
- 範例：方文山〈髮如雪〉中的時空對話感

#### 模式 D：第三人稱觀察（冷靜疏離）

- 從頭到尾不出現「我」
- 像在描述一個故事
- 適合：社會議題、寓言式歌曲
- 範例：陳昇〈把悲傷留給自己〉部分段落

### 視角切換時的訊號

當切換視角時，**第一句必須有「視角切換訊號詞」**，否則聽眾會迷失：

| 切換方向 | 訊號詞範例 |
|---|---|
| → 第二人稱 | 「妳呢⋯」「告訴我⋯」「你還記得⋯」 |
| → 第三人稱 | 「他們說⋯」「這座城市⋯」「有人⋯」 |
| → 時空閃回 | 「那年⋯」「曾經⋯」「如果時間⋯」 |
| → 抽離全景 | 「世界⋯」「時間⋯」「人生⋯」 |

## 6. 七感感官調用

### 核心原則

普通寫作只用視覺，**好歌詞會調用 7 個感官通道**。每段歌詞至少觸發 2 個以上感官。

### 七感清單與作詞應用

| 感官 | 範圍 | 作詞範例 |
|---|---|---|
| **視覺** | 顏色、形狀、光線、動作 | 「夕陽染紅了窗簾」 |
| **聽覺** | 聲音、寂靜、音調 | 「電話那頭只剩呼吸聲」 |
| **嗅覺** | 氣味、香、臭 | 「你毛衣上殘留的菸味」 |
| **味覺** | 酸甜苦辣鹹、口感 | 「眼淚是鹹的 你說不算」 |
| **觸覺** | 質感、溫度、痛癢 | 「他的手心比冬天更冷」 |
| **有機感** | 身體內部感受：心跳、暈眩、噁心、飢餓、氣促 | 「胸口像被什麼堵住」 |
| **動覺** | 運動感、平衡、空間關係 | 「整個世界都在旋轉」「我朝你走了三步又退回來」 |

### 進階：七感的「層次組合」

#### 表層組合（直接描述）
「夕陽染紅了窗簾」（視）「飯菜的香味」（嗅）「他不在家」（事實）

#### 深層組合（多感聯動）
「夕陽（視）染紅了空蕩的餐桌（視 + 空間感），飯菜的香味（嗅）還在 但你的椅子是冷的（觸 + 有機感）」

→ 4 個感官同時觸發，畫面密度極高

### 抽象情緒 → 七感具體化對照表

| 抽象情緒 | 七感具體化範例 |
|---|---|
| 想念 | **嗅**「衣櫃裡你的香水還沒散」**動覺**「我朝電話走了三步又退回來」 |
| 焦慮 | **有機感**「胃像有人在裡面擰毛巾」**動覺**「我來回踱步把地板踩出聲」 |
| 釋懷 | **觸**「風吹過耳邊不再冷」**聽**「我終於聽見鳥叫」 |
| 孤獨 | **聽**「冰箱嗡嗡的聲音變得很大」**視**「公寓的燈只開了一盞」 |
| 心動 | **有機感**「心跳得我自己都聽見」**觸**「手心開始冒汗」 |
| 失落 | **動覺**「身體變得很重 抬不起頭」**味**「咖啡今天特別苦」 |

### 訓練法：感官回憶日記

每天 5 分鐘，回想當天最有感的一個時刻：

1. 那時候**看到**什麼？
2. **聽到**什麼？
3. **聞到**什麼？
4. 身體**內部**有什麼感覺（心跳 / 胸悶 / 暈 / 緊繃）？
5. **動作**上你做了什麼（後退 / 前傾 / 握緊 / 鬆手）？

把這 5 點寫下來 → 累積成「個人感官庫」→ 寫詞時直接調用。

## 7. 整合工作流

執行創作時，**依照下方順序操作**，不要跳步：

### 步驟 1：素材準備（Pre-composition）

- 從使用者 Premise 提取關鍵詞 3-5 個
- 每個關鍵詞做一次「七感擴展」（見[§6 七感感官調用](#6-七感感官調用)）
- 累積出 15-20 個具體意象 / 物件 / 動作備用

### 步驟 2：Hook 鎖定

- 依照 Q2 選擇的方向（A/B/C/D），產出 5-10 句候選 Hook
- 用「四項判定」（見[§2 鉤子優先](#2-鉤子優先hook-first)）篩出 1 句最強 Hook
- 確認 Hook 的 Prosody（重音、長音、發音飽和度）

### 步驟 3：結構規劃

- 決定段落順序（標準：intro / verse / pre-chorus / chorus / verse / pre-chorus / chorus / bridge / chorus / outro）
- 規劃視角切換點（見[§5 視角切換推進](#5-視角切換推進)）
- 規劃韻部分配（見[§3 韻腳張力管理](#3-韻腳張力管理)）

### 步驟 4：填詞（Composition）

- 從副歌（Chorus）開始寫（Hook First）
- 反向設計主歌 1（鋪陳通往 Hook 的「之前」）
- 設計 Pre-chorus（張力過渡）
- 寫主歌 2（視角切換 + 情境推進）
- 寫橋段（換韻、抽離視角、為最終副歌蓄力）
- 寫 Intro / Outro（氛圍鋪陳與餘韻）

### 步驟 5：自檢（Self-review）

依照鐵則清單檢查：

- [ ] 是否每段都有具體畫面？
- [ ] 抽象詞是否都伴隨具體物件？
- [ ] 韻腳是否有主韻 + 變韻？
- [ ] 視角是否有切換？
- [ ] 七感是否有觸發 2 個以上？
- [ ] Hook 是否唸出來順？
- [ ] 是否有撞韻、擠韻、倒字？

### 步驟 6：三變體差異化

- **版本 1（標準版）**：嚴格按使用者選擇
- **版本 2（對位版）**：Hook 方向換成相反的（例：選 A 改 B）
- **版本 3（實驗版）**：結構或視角大膽嘗試
  - 時間倒敘
  - 雙視角對話
  - 書信體
  - 第三人稱旁觀
  - 對白嵌入歌詞
